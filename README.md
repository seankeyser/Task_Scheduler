# Task Scheduler Web App

A multi-page task management app built using HTML, CSS, and JavaScript. This project allows you to:

- Add detailed tasks
- View all tasks with priority and due date
- Mark tasks as completed
- Store tasks using browser localStorage

## Pages

- `index.html`: Home page
- `add.html`: Add a new task
- `view.html`: View and manage tasks

## Deployment

This app is ready for deployment via GitHub Pages.

1. Upload all files to a public GitHub repository.
2. Go to **Settings > Pages**.
3. Choose branch `main` and root `/`.
4. Save and wait for the live link.
